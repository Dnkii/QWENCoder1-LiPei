# 保险理赔助手系统

这是一个基于大模型技术的保险理赔辅助系统，能够自动处理理赔文档和影像资料，实现智能分类、关键信息提取和责任判定。

## 系统功能

1. 文档/影像智能分类
2. 关键信息提取
3. 基于保险合同的责任判定

## 目录结构

```
insurance_claim_assistant/
├── docs/           # 文档目录
├── src/            # 源代码目录
├── tests/          # 测试代码目录
├── config/         # 配置文件目录
├── data/           # 数据目录
└── README.md       # 项目说明文件
```

## 技术架构

- 大语言模型：用于文档理解和信息提取
- 分类算法：对文档类型进行识别
- NLP技术：关键词提取和语义分析
- 规则引擎：基于合同条款进行责任判定
